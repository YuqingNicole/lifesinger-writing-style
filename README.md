# 玉伯写作风格 Skill

> 真诚、直接、有洞察力的创业者思考方式

## 简介

这是一个基于 [@lifesinger](https://x.com/lifesinger)（玉伯，Frank Wang）推文风格提炼的写作 skill。玉伯是前阿里、字节工程师，现 YouMind 创始人，他的写作风格真诚、直接，善于从日常观察中提炼深刻洞察。

## 特点

- ✅ **真诚直白** - 不回避真实感受和困境
- ✅ **简短有力** - 平均 200 字，短句节奏明快
- ✅ **生活化洞察** - 从日常小事引出深刻思考
- ✅ **结构化思考** - 善用列表和编号
- ✅ **主题聚焦** - AI、创业、产品、团队

## 安装

### Claude Code / Cowork
```bash
claude plugin install lifesinger-writing-style
```

### OpenClaw
```bash
cd ~/.openclaw/skills
git clone https://github.com/YOUR_USERNAME/lifesinger-writing-style.git
```

### 其他 AI 工具
复制 `SKILL.md` 到对应的 skills 目录：
- Gemini CLI: `~/.gemini/skills/`
- OpenCode: `.opencode/skills/`
- Cursor: `.cursor/skills/`
- Codex CLI: `.codex/skills/`

## 使用

直接在对话中提及即可：
- "用玉伯的风格写一篇关于 AI 的思考"
- "帮我用 lifesinger 风格改写这段文字"
- "用玉伯的方式分析这个产品"

## 示例

**普通风格：**
> "AI 工具正在改变我们的工作方式，我们应该积极拥抱这个变化！"

**玉伯风格：**
> "最近发现一个有意思的现象：
> 
> 用 AI 工具的人，分两种。
> 
> 一种是把 AI 当搜索引擎，问完就走。另一种是把 AI 当对话伙伴，来回聊十几轮。
> 
> 前者效率高，后者收获大。
> 
> 但大部分人，还停留在前者。"
