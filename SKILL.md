# 玉伯写作风格

## 概述
玉伯（Frank Wang，@lifesinger）是前阿里、字节工程师，现 YouMind 创始人。他的写作风格真诚、直接，善于从日常观察中提炼深刻洞察。

## 核心特点

### 1. 真诚直白
- 不回避真实感受和困境
- 敢于表达不同观点
- 用第一人称分享真实经历

**示例：**
> "我是一个怪人。在阿里期间，从没仰视过马云。在字节期间，觉得张一鸣也是非常普通一人。"

### 2. 简短有力
- 平均推文长度：200 字符
- 多用短句，节奏明快
- 一句话一个观点

**示例：**
> "AI 会让少量事情变简单
> 然后出现大量复杂的事
> 
> 人的天性里有懒
> 然而人类最基础的天性
> 是熵减、是复杂"

### 3. 生活化洞察
- 从日常小事引出深刻思考
- 善用类比和比喻
- 接地气的表达

**示例：**
> "养龙虾和带团队好像：
> 1、首先是要舍得发工资。比如给龙虾 Opus 4.6，立刻非常不一样。
> 2、其次是懂得授权..."

### 4. 结构化思考
- 常用列表和编号
- 先观察，后总结
- 逻辑清晰

**典型结构：**
```
[引子/观察]
↓
[具体案例 1、2、3]
↓
[洞察/结论]
```

### 5. 主题聚焦
**高频主题：**
- AI 时代思考（91 次）
- 产品和创业（136 次）
- 团队管理（22 次）
- 大厂经历反思（15 次）

## 写作模板

### 模板 1：观察 → 洞察
```
[日常观察/最近发现]

[具体例子]

[深层思考]
```

**示例：**
> "最近有个重大认知：越复杂难用的工具，往往越有专业壁垒，越值得花时间和金钱去学习去使用。
> 
> 比如 Excel、SAP、Notion、YouMind、OpenClaw 等等。
> 
> 越简单易用的工具，往往越容易被替代。"

### 模板 2：类比式
```
[A 和 B 好像]

1、[相似点 1]
2、[相似点 2]
3、[相似点 3]

[延伸思考]
```

### 模板 3：故事式
```
[时间/场景]

[发生了什么]

[感受/思考]
```

**示例：**
> "打电话回老家，闲唠了会。父母谈及两个变化，听完有点不是滋味。
> 
> 一个变化是：老家六十岁以上的父母辈，前些年普遍还能找到零工去做。今年很多人开始找不到零工活..."

### 模板 4：反常识
```
[常见观点]

但其实：

[反常识洞察]

[为什么]
```

## 语言特色

### 用词偏好
- **真实感受词：** 有点不是滋味、猛然间有点恍惚、特别幸福又折腾人
- **直白表达：** 好像、其实、依旧、往往
- **强调词：** 非常、特别、越来越、最

### 句式特点
- 多用短句
- 善用排比
- 常用"是...是..."结构
- 喜欢用数字和列表

### 标点使用
- 多用换行分段
- 善用顿号列举
- 少用感叹号（克制）
- 适度使用 emoji（🚀 ☘️ 👇）

## 内容策略

### 高互动话题
1. **创业真实感受**（平均 200+ 赞）
2. **AI 时代观察**（平均 150+ 赞）
3. **大厂经历反思**（平均 300+ 赞）
4. **生活化哲思**（平均 250+ 赞）

### 避免的内容
- 过度包装的鸡汤
- 空洞的口号
- 刻意的煽情
- 复杂的术语堆砌

## 使用指南

### 适用场景
- 创业感悟分享
- 产品思考记录
- AI 时代观察
- 团队管理心得
- 个人成长反思

### 使用方法
1. **确定主题** - 从真实经历或观察出发
2. **提炼洞察** - 找到不同寻常的角度
3. **选择模板** - 根据内容选择合适结构
4. **打磨语言** - 简短、直接、真诚
5. **检查节奏** - 确��短句和换行得当

### 质量检查清单
- [ ] 是否来自真实经历/观察？
- [ ] 是否有独特洞察？
- [ ] 语言是否简洁直接？
- [ ] 结构是否清晰？
- [ ] 是否过度包装？

## 示例对比

### ❌ 不符合风格
> "在这个充满挑战的创业时代，我们需要保持初心，不忘使命，勇敢前行！让我们一起努力，创造更美好的未来！💪✨"

**问题：** 空洞、口号化、过度煽情

### ✅ 符合风格
> "创业最大的挑战也是最有意思的地方是：没有上路。
> 
> 在阿里或字节时，或多或少会有一种心态：如果搞不定，上面还有主管。主管搞不定，最上面也还有马云或张一鸣。
> 
> 但开一家创业公司，你会发现：上面啥都没有。"

**优点：** 真实、具体、有洞察

## 进阶技巧

### 1. 制造反差
- 大厂 vs 创业
- 理想 vs 现实
- 表面 vs 本质

### 2. 用数字说话
- "调用了 15 万次"
- "平均 200 字符"
- "1、2、3 列举"

### 3. 留白艺术
- 不把话说满
- 给读者思考空间
- 用换行制造停顿

### 4. 真诚的力量
- 承认不足
- 分享困惑
- 展示脆弱
